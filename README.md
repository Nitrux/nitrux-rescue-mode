# Nitrux Rescue Mode

![](https://raw.githubusercontent.com/Nitrux/luv-icon-theme/master/Luv/apps/64/system-help.svg)

Rescue Mode is a boot path implemented through GRUB and initramfs for restoring a NUTS backup by following the concept of a "Self-Healing Initramfs".
